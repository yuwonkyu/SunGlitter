# 윤슬이집 스튜디오 예약 서비스 안내

이 문서는 윤슬이집 스튜디오 운영을 위해 만든 예약 서비스의 사용 목적과 운영 포인트를 쉽게 확인하기 위한 안내서입니다.

## 이 서비스로 할 수 있는 일

- 손님은 예약 안내 페이지와 예약 현황 페이지를 확인할 수 있습니다.
- 운영자는 관리자 페이지에서 날짜별 예약 슬롯을 등록, 수정, 삭제할 수 있습니다.
- 예약 상태는 예약 가능, 대기, 예약 완료로 관리됩니다.
- 예약 시간은 30분 단위로만 등록됩니다.

## 손님이 보는 화면

- 메인 페이지: 예약 안내, 예약 현황, 인스타그램, 지도 링크를 한 번에 확인
- 예약 안내 페이지: 예약 절차와 문의 방법 확인
- 예약 현황 페이지: 달력으로 날짜별 예약 상태 확인

## 관리자 페이지 안내

- 관리자 페이지 주소는 보안상 이 문서에 전체를 적지 않습니다.
- 주소 형식 예시: `/***************-******-***/`
- 실제 접속 주소와 비밀번호는 별도 전달된 정보만 사용해 주세요.
- 관리자 페이지는 비공개 주소 + 로그인 확인 단계를 거치도록 구성되어 있습니다.

## 최근 정리한 내용

- 예약 데이터 검증 방식을 정리해 입력 오류가 생겼을 때 더 안정적으로 막도록 수정했습니다.
- 관리자 화면과 달력 화면에서 불필요한 다시 그리기를 줄여 반응성을 다듬었습니다.
- 단순한 화면 요소에는 과한 최적화를 걷어내어, 유지보수성과 안정성을 함께 맞췄습니다.

## 예약 데이터 저장 방식

- 기본적으로는 외부 저장소에 예약 데이터가 저장됩니다.
- 외부 저장소 연결 정보가 없으면 서버 안의 임시 메모리에 저장됩니다.
- 임시 메모리 저장 상태에서는 서버를 다시 켜면 예약 데이터가 사라질 수 있습니다.

즉, 실제 운영에서는 외부 저장소 연결이 유지되어야 예약 정보가 안전하게 보관됩니다.

## 운영 시 체크할 점

- 예약 등록 후에는 예약 현황 페이지에서 실제로 보이는지 확인해 주세요.
- 관리자 비밀번호와 관리자 접속 주소는 외부에 공유하지 않는 것이 좋습니다.
- 장기간 운영 시에는 예약 데이터 백업 여부를 주기적으로 확인하는 것이 안전합니다.

## 참고 메모

- 이 서비스는 윤슬이집 스튜디오 운영 목적에 맞춰 제작되었습니다.
- 별도 승인 없이 다른 서비스로 복제하거나 재사용하는 것은 허용되지 않습니다.
